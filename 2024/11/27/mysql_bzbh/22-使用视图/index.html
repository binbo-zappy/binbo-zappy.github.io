22. 使用视图

1. 视图

  1. 视图是虚拟的表。与包含数据的表不一样，视图只包含使用时动态检索数据的查询。
  2. 为了创建视图，必须具有足够的访问权限。这些限制通常由数据库管理人员授予
  3. ORDER BY可以用在视图中，但如果从该视图检索数据SELECT中也含有ORDER BY，那么该视图中的ORDER BY将被覆盖。

2. 使用视图

  1. 视图用CREATE VIEW语句来创建。
  2. 使用SHOW CREATE VIEW viewname；来查看创建视图的语句
  3. 用DROP删除视图，其语法为DROP VIEW viewname;。
  4. 更新视图时，可以先用DROP再用CREATE，也可以直接用CREATE OR REPLACE VIEW。
